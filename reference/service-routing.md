# service-routing.md — Patterns, Tiers, Pricing

The studio prescribes; the client doesn't pick from a menu. Route the diagnostic to a pattern A–H, set the tier, land one specific price, draft the proposal. If the right prescription differs from what they asked for, prescribe the real thing and say why — if they reject it and demand the original ask, that's a fit problem → **decline** (`vendor-labor`).

---

## Prescription patterns A–H (verbatim)

| Key | Diagnostic surfaces → Prescription | Tier | Price |
|---|---|---|---|
| **A — Small business needs a website** | Existing business, no/terrible site, single DM → **Websites** | Standard | $2,500–5,000 (1 wk) |
| **B — Local service business wants more leads** | Established service biz, basic site, weak lead flow → **Customer acquisition (Retainer) + light Marketing strategy upfront** | Retainer after Standard setup | $2,500 setup, then $1,000–2,500/mo |
| **C — Founder wants a personal brand site** | Individual w/ body of work, no site, wants to publish → **Websites + Content (initial copy + first essays)** | Standard | $3,500–5,000 |
| **D — E-commerce brand needs upgrade** | Existing DTC w/ revenue, dated/template site → **Websites (Build) + Branding refresh + Content** | Build | $8,000–12,500 |
| **E — Operator wants to add AI** | Existing biz, established workflows, wants AI augmentation → **AI system builds — ICM workspace install** | Build | $5,000–10,000 |
| **F — Multi-element new business launch** | Founder launching, needs the whole stack → **Branding + Websites + Content + AI (phased Build)** | Build, phased | $10,000–25,000 across two engagements |
| **G — Existing business needs everything** | Established biz, everything rotted → **Phase 1: Branding + Websites (Build); Phase 2: Content + Acquisition (Retainer)** | Build → Retainer | $20,000–40,000 first-year |
| **H — Small content asset, fast** | Specific deliverable, fast turnaround, contained scope → **Single Express engagement** | Express | $750–2,000 |

**Common misdiagnoses** (prescribe the real thing): asks for "a website" → needs **branding first** (incoherent identity); asks for "more leads" → needs **better positioning**; asks for "AI" → needs **operations consulting first** (broken workflows); asks for "branding refresh" → has a coherent brand, needs **marketing**; asks for "custom software" → needs a **better workflow or AI tools** (10x cheaper).

## The 4 tiers

| Tier | Range | Timeline |
|---|---|---|
| **Express** | $750–2,000 | 1–3 days |
| **Standard** | $2,500–5,000 | 1 week |
| **Build** | $5,000–12,500 | 2 weeks |
| **Retainer** | $1,000–2,500 / month | recurring |

**No Standard under $2,500.** If adjustments push above the ceiling, it's the next tier up (or phase it / custom-quote $15–30K / refer enterprise to Substrate). If below the floor, it's the next tier down or declined.

## Pricing decision tree — land ONE number

Never quote a range to a client. Five inputs, then walk it.

**Inputs:** (1) **Scope** — heaviest; more deliverables = higher. (2) **Complexity** — second heaviest; custom functionality/integrations/strict constraints push to ceiling. (3) **Client size/budget** — medium; small pays floor, large pays ceiling for the same work. (4) **Strategic value** — light; case-study/referral potential subtracts. (5) **Practitioner conviction** — light; lower conviction prices toward ceiling (covers risk).

**The walk-through:**
1. **Start at the tier midpoint** — Express **$1,375** · Standard **$3,750** · Build **$8,750** · Retainer **$1,750**.
2. **+** for scope — each significant extra deliverable +10–20% of the range.
3. **+** for complexity — custom functionality/integrations/design difficulty +10–20%.
4. **−** for strategic value — case-study or referral likelihood −5–15%.
5. **±** for client size — read their budget signals.
6. **Round clean** — end in $0 or $500, never $3,250.

Final must sit inside the tier range. Above ceiling → next tier / phase / custom. Below floor → next tier down or decline. *Worked example: Standard restaurant site — midpoint $3,750, small local client −$500, low strategic value +$250 → **$3,500**.*

If a client pushes on price: not "we can do it for less" — **"what would you take out of the scope to bring it down?"** Scope flexes; price doesn't.

## TIER_MILESTONES — payment splits

| Tier | Milestones |
|---|---|
| **Express** | 100% upfront |
| **Standard** | 50% on signing · 50% on delivery |
| **Build** | 50% on signing · 25% at midpoint · 25% on delivery |
| **Retainer** | Monthly (per cycle) |

On `proposal → won`, draft the kickoff + the first invoice/payment ask using this split. Proposal price-line format: *"Total fee: $4,500. Payment: 50% on signing ($2,250), 50% on delivery ($2,250). Stripe link sent with the signed letter."*
